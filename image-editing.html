Alight Motion: Image Editing

Did you know that Alight Motion can also be used to edit images or design something in an image format?

On this page, You can learn that.

 
 
 

Editing Images

In the alight motion app, just import an image. Then, you can use the effects tab and other shapes to edit an image easily, so you don't have to care about any keyframes, or timing here, because you're editing a picture.

If instructions unclear in the videos, subtitles with instructions are available.

1. Import an image.

You can repeat this step multiple times. Don't import too much! Alight Motion is intended and optimized for shapes and vectors, rather than media.

2. Adding effects

You can repeat this step multiple times, there is (almost) no limit to effects on an object.

3. Exporting and removing the WATERMARK

You should be all done after this step. Exporting plays a 30 second ad, so make sure to watch it, because skipping it right away won't allow the video to export.
